## Sobre

O **Travelgram** é uma página estática de perfil de viagens desenvolvida com HTML e CSS.

O projeto apresenta informações de uma viajante, como localização, quantidade de países visitados, número de fotos e uma galeria com registros das viagens.

Além da construção do layout original, o projeto foi adaptado para diferentes tamanhos de tela, proporcionando uma melhor experiência de navegação em dispositivos móveis, tablets e desktops.

## 📱 Responsividade

A interface foi desenvolvida para se adaptar a diferentes larguras de tela.

Durante a implementação da responsividade, foram trabalhados conceitos como:

- Media queries
- Breakpoints definidos de acordo com o comportamento do conteúdo
- Alteração da direção dos elementos com Flexbox
- Reorganização do menu de navegação no mobile
- Adaptação da seção de perfil
- Reposicionamento das informações do usuário
- Galeria responsiva
- Imagens com proporção controlada
- Ajustes de espaçamento para diferentes dispositivos
- Prevenção de rolagem horizontal

No mobile, os elementos do perfil são organizados verticalmente, os links menos importantes do menu são ocultados e as imagens da galeria se adaptam ao espaço disponível.

## Como executar

1. Clone este repositório ou copie os arquivos para sua máquina.
2. Abra o arquivo `index.html` no navegador.
3. Opcionalmente, utilize a extensão Live Server no VS Code para visualizar o projeto localmente.

## Estrutura de pastas

- `index.html` — página principal
- `styles/index.css` — arquivo principal responsável pelas importações
- `styles/global.css` — estilos globais, variáveis e container
- `styles/nav.css` — estilos do menu de navegação
- `styles/header.css` — estilos da seção de perfil
- `styles/main.css` — estilos da galeria de imagens
- `styles/footer.css` — estilos do rodapé
- `assets/Logo.svg` — logotipo do projeto
- `assets/icons/` — ícones utilizados na interface
- `assets/images/` — imagens do perfil e da galeria

## Funcionalidades

- Menu de navegação responsivo
- Seção de perfil com foto, nome e descrição
- Informações de localização, países visitados e quantidade de fotos
- Galeria de imagens
- Layout adaptado para mobile, tablet e desktop
- Imagens responsivas com proporção preservada
- Reorganização automática dos elementos conforme a largura da tela
